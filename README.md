This is an example CLI built using [oclif](https://github.com/oclif/oclif).  It outputs [kaomoji](https://en.wikipedia.org/wiki/Kaomoji) from words.

```
$ ./bin/run shrug
¯\_(ツ)_/¯

$ ./bin/run happy
ᕕ( ᐛ )ᕗ

$ ./bin/run tableflip
(ノ ゜Д゜)ノ ︵ ┻━┻
```
